settype

settype -- Set the type of a variable.

Description

int settype(string var, string type);

Set the type of variable var to type.

Possibles values of type are:

  • "integer"

  • "double"

  • "string"

  • "array"

  • "object"

Returns true if successful; otherwise returns false.

See also gettype().
