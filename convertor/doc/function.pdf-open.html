PDF_open

PDF_open -- Opens a new pdf document

Description

int pdf_open(int file, int info);

The PDF_open() function opens a new pdf document. The corresponding file has to be opened with fopen() and the file descriptor passed as argument file. info is the an info structure that has to be created with pdf_get_info(). The info structure will be deleted within this function.

Note: The return value is needed as the first parameter in all other functions writing to the pdf document.

See also fopen(), PDF_get_info().
